## 함수

- xx.isdigit() -> 문자열을 대상으로 사용하는 함수이며, 해당 문자열의 전체 구성이 숫자인지 문자인지를 bool 형태로 반환
  
  - a = '123456'
  
  - a.isdigit() = True
  
  - b = '123abc'
  
  - b.isdigit() = False
  
  - b[0:2].isdigit() = True
